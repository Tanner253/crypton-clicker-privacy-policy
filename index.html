Privacy Policy

Effective Date: May 15, 2024

Policy Update Notice: This privacy policy has been updated to include information about our in-game chat functionality. Please review the changes below.

Overview

Crypton Clicker collects minimal data to provide core functionality and now includes an optional in-game chat system. This policy outlines what information is collected and how it is used.

Data Collection

We collect the following limited information:

  • Device Identifier: A randomly generated identifier stored on your device to support chat functionality and local game saves
  • Chat Messages: Messages you send in the in-game chat system are temporarily stored to enable communication between players
  • Username: If you choose to set a custom username for the chat feature, this is stored locally on your device
  • Game Progress Data: Your game progress is saved locally on your device

We do not collect any of the following:

  • Personal information (name, email, address, etc.)
  • Location data
  • Browsing history
  • Any data from third-party SDKs or APIs not specified in this policy

Chat Functionality

The in-game chat system follows these privacy and safety principles:

  • Content Moderation: All chat messages are filtered to remove inappropriate content
  • Data Storage: Chat messages are stored temporarily to facilitate communication and are not permanently retained
  • User Control: The chat feature can be disabled at any time through game settings
  • Children's Safety: Additional safeguards are implemented to protect users under 13 years of age in compliance with COPPA
  • Reporting: Users can report inappropriate messages or behavior through an in-game reporting system

Third-Party Services

This game is built using Unity 6 and published on the Google Play Store. While Unity and Google may have their own policies, Crypton Clicker only uses the following third-party services:

  • Firebase Realtime Database: Used to securely store and transmit chat messages (when chat functionality is enabled)
  • Firebase Authentication: Used to generate anonymous user identifiers for chat functionality (no personal information required)

These services operate under their own privacy policies and comply with GDPR, CCPA, and other relevant privacy regulations.

Data Security

Chat messages and user identifiers are encrypted during transmission and storage to protect user privacy. We implement reasonable security measures to protect the limited data we collect.

Children's Privacy

Crypton Clicker complies with the Children's Online Privacy Protection Act (COPPA). We do not knowingly collect personal information from children under 13. The chat feature includes enhanced protections for younger users, including stricter content filtering and limited interaction options.

Future Changes

If we decide to update the game to include any form of additional data collection (for analytics, crash reports, or otherwise), this privacy policy will be updated and users will be notified accordingly.

Your Rights

You have the right to:

  • Disable the chat functionality at any time
  • Request deletion of your device identifier and any associated chat data
  • Report concerns about privacy or data handling
